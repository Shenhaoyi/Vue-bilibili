# bilibili
仿b站移动端开发
技术栈 vue + vue-router + axios + sass + vant

### 预览
建议使用浏览器的手机模式访问。

**预览必看**：若浏览器自动重定向到 https 的协议，可以使用无痕模式访问！

预览链接：http://shenhaoyi.com/vue-bilibili-web/
